**FishMark** — the outline-fish mascot (摸鱼); use it as the system's logo bug, a bullet, or a section-ending flourish.

```jsx
<FishMark size={80} color="var(--ink)" />
<FishMark size={48} color="var(--red)" flip />
```

Monoline only — never fill it. Recolor via `color` (navy ink by default; red works for high-register/accent contexts). Height is derived from width automatically.
