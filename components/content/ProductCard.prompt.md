**ProductCard** — a branded "appliance"/derivative tile: a prop illustration on a flat mint/coral field with a name and one deadpan line.

```jsx
<ProductCard
  image="assets/illustrations/objects/cup.png"
  name="Mini teacups"
  description="A great tool for Cyberloafing during work days."
  field="var(--mint-wash)"
/>
```

Use the copied prop PNGs from `assets/illustrations/objects/`. This is the only component that carries a soft shadow — keep everything else flat.
