**RegisterTag** — the depth-of-loafing chip (浅/中/高 摸鱼), color-coded #EBC3C6 → coral → red for low → moderate → high risk, white text throughout.

```jsx
<RegisterTag level="light" />
<RegisterTag level="mid" />
<RegisterTag level="high" showEn={false} />
```

Use one per technique. The color mapping (#EBC3C6/coral/red) is fixed — don't recolor it.
